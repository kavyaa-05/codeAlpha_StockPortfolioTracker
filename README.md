## 📌 Project Title

**Stock Portfolio Tracker (CodeAlpha Task 2)**


## 📖 Description
This project is a simple console-based Stock Portfolio Tracker developed as part of the CodeAlpha internship. It allows users to enter stock names and their quantities, then calculates the total investment value using predefined stock prices.
The project demonstrates how basic programming concepts can be applied to real-world scenarios like financial tracking and portfolio management.


## ✨ Features
* 📊 User input for stock names and quantities
* 📈 Predefined stock prices using a dictionary
* 🧮 Calculates total investment value
* 🔁 Supports multiple stock entries (if implemented)
* ⚡ Simple and interactive console-based interface
* 💾 Option to save results to a `.txt` or `.csv` file (if implemented)


## ▶️ How to Run the Project
1. Make sure Python is installed on your system
2. Clone this repository or download the files
3. Open the project folder in VS Code or any IDE
4. Run the file using:
python portfolio.py
5. Enter stock name and quantity when prompted
6. View your total investment value
